## Cybersecurity & DevSecOps Interest

Security isn't an afterthought in how I think about engineering — it's part of the design.

| Area | What draws me in |
|------|------------------|
| **Security mindset** | Threat awareness, least privilege, and building systems that fail safely |
| **Infrastructure automation** | Repeatable environments with Docker, scripts, and IaC principles |
| **CI/CD pipelines** | Automated testing, linting, and security checks in the delivery flow |
| **Cloud security** | Identity, network boundaries, secrets management, and secure deployments |

I'm learning to connect these areas so security is embedded early — not bolted on at the end.
